# دليل الاستخدام السريع - نظام التنقل بين العيادات

## 🚀 البدء السريع

### 1️⃣ لا شيء لتفعله! 
النظام يعمل **تلقائياً** عند تشغيل التطبيق:
- يتم تحميل جميع العيادات المتاحة تلقائياً
- يتم عرض خيار "تبديل العيادة" في القائمة الجانبية (إذا كانت هناك أكثر من عيادة)

---

## 🎮 طرق التنقل بين العيادات

### الطريقة 1️⃣: من القائمة الجانبية (الأسهل!)
```
1. اسحب من اليسار أو انقر على فتح القائمة
2. ابحث عن "تبديل العيادة"
3. اختر العيادة المطلوبة
4. سيتم التبديل والتحميل تلقائياً ✨
```

### الطريقة 2️⃣: من الكود
```dart
final clinicService = ClinicService();

// التبديل إلى عيادة معينة
await clinicService.switchToClinic(clinic);

// أو التبديل السريع للعيادة التالية
await clinicService.switchToNextClinic();
```

### الطريقة 3️⃣: إضافة أزرار سريعة في AppBar
```dart
QuickClinicNavigation(
  onClinicChanged: () {
    reloadAllData();
  },
)
```

## 📱 معلومات العيادات

### معلومات يتم عرضها:
- ✅ اسم العيادة
- ✅ تاريخ آخر وصول (مثل: "قبل ساعتين")
- ✅ علامة تشير إلى العيادة الحالية

### اسم العيادة:
يتم استخراجه من اسم ملف قاعدة البيانات:
```
clinic_name.db  →  clinic_name
roni.db         →  roni
main_clinic.db  →  main_clinic
```

## ❓ الأسئلة الشائعة

### S: لماذا لا أرى خيار "تبديل العيادة"؟
**J:** لأن لديك عيادة واحدة فقط. هذا الخيار يظهر فقط عندما يكون لديك عيادتان أو أكثر.

### S: هل يتم حفظ تفضيلاتي عند التبديل؟
**J:** نعم! يتم حفظ:
- اسم العيادة الحالية
- تاريخ آخر وصول
- جميع البيانات المحلية

### S: ماذا يحدث للبيانات القديمة؟
**J:** لا شيء! كل عيادة لها قاعدة بيانات منفصلة، بياناتها آمنة تماماً.

### S: هل يمكنني حذف عيادة؟
**J:** نعم، من خلال صفحة **الإعدادات** → **إدارة البيانات**
⚠️ **تحذير:** لا يمكن حذف العيادة الحالية (يجب التبديل أولاً)

### S: كيف أنشئ عيادة جديدة؟
**J:** من صفحة **الإعدادات** → **إدارة البيانات** → **إنشاء عيادة جديدة**

## 🔍 التحقق من الحالة

### عرض معلومات العيادات:
```dart
final clinicService = ClinicService();

// عدد العيادات
print('عدد العيادات: ${clinicService.clinicsCount}');

// اسم العيادة الحالية
print('العيادة الحالية: ${clinicService.currentClinicName}');

// قائمة جميع العيادات
for (var clinic in clinicService.clinics) {
  print('📍 ${clinic.name} - آخر وصول: ${clinic.lastAccessedAt}');
}

// ملخص العيادات
print(clinicService.getClinicsSummary());
```

## 🐛 استكشاف الأخطاء

### المشكلة: لم يتم تحديث البيانات بعد التبديل
**الحل:** تأكد من استدعاء `reloadAllData()`:
```dart
await clinicService.switchToClinic(clinic);
await reloadAllData(); // ✅ مهم جداً
```

### المشكلة: لا يمكن حذف عيادة
**الحل:** تبديل إلى عيادة أخرى أولاً:
```dart
if (clinic.dbFileName == selectedDbName) {
  // العيادة الحالية - لا يمكن حذفها
  await clinicService.switchToNextClinic();
}
await clinicService.deleteClinic(clinic);
```

### المشكلة: الأداء بطيء
**الحل:** 
1. تأكد من أن قاعدة البيانات ليست كبيرة جداً
2. قم بحذف قواعد البيانات القديمة غير المستخدمة
3. استخدم نسخ احتياطية
