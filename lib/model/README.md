## 💡 أمثلة الاستخدام

### إنشاء عيادة جديدة
```dart

  name: 'عيادة نور',
  dbFileName: 'clinic_noor.db',
);
```

### إنشاء مريض
```dart
PatientModel patient = PatientModel(
  name: 'أحمد محمد',
```
