# 🎨 واجهات التنقل بين العيادات - Clinic Switcher Widgets

## 📖 نظرة عامة
يحتوي هذا المجلد على واجهات المستخدم للتنقل السريع بين العيادات.

---

## 📦 الملفات الموجودة

### 1. `ClinicSwitcher.dart` - 🌟 واجهات التبديل

#### الفئات الرئيسية

##### **أ) `ClinicSwitcher` StatefulWidget**
واجهة متعددة الأوجه للتنقل بين العيادات.

**الخصائص:**
- `onClinicChanged` - callback عند تغيير العيادة
- `showAsIcon` - نوع العرض (أيقونة أم مفصل)

**الأنواع:**

1. **Icon View** - عرض كأيقونة منسدلة
```
🏢 ▼ 
  ├─ عيادة نور ✓
  ├─ عيادة الزهراء
  └─ عيادة النور 2
```

2. **Detailed View** - عرض مفصل مع التفاصيل
```
العيادات (3)
┌─────────────────────┐
│ 🏢 عيادة نور        │ ✓
│   آخر وصول: للتو    │
├─────────────────────┤
│ 🏢 عيادة الزهراء    │
│   آخر وصول: أمس     │
└─────────────────────┘
```

**الدوال:**
- `_buildIconView()` - بناء الأيقونة
- `_buildDetailedView()` - بناء العرض المفصل
- `_loadClinics()` - تحميل البيانات
- `_switchClinic()` - تبديل العيادة

##### **ب) `QuickClinicNavigation` StatefulWidget**
واجهة تنقل سريع مع أزرار التالي/السابق.

**الشكل:**
```
◀ عيادة الحالية ▶
```

**المميزات:**
- تنقل سريع بدون قوائم
- عرض اسم العيادة الحالية
- أيقونات توضيحية

---

## 💡 أمثلة الاستخدام

### الاستخدام الأساسي
```dart
ClinicSwitcher(
  showAsIcon: true, // عرض كأيقونة
  onClinicChanged: () {
    reloadAllData(); // إعادة تحميل البيانات
  },
)
```

### في AppBar
```dart
AppBar(
  title: const Text('الرئيسية'),
  actions: [
    QuickClinicNavigation(
      onClinicChanged: () {
        reloadAllData();
      },
    ),
  ],
)
```

### في القائمة الجانبية
```dart
ClinicSwitcher(
  showAsIcon: false, // عرض مفصل
  onClinicChanged: _onClinicChanged,
)
```

## 🔄 تحديث البيانات

بعد كل تبديل عيادة، يجب استدعاء:
```dart
await reloadAllData();
```

- ✅ تحميل بيانات العيادة الجديدة
- ✅ تحديث المرضى والموظفين

## 🐛 استكشاف الأخطاء

| المشكلة | السبب | الحل |
